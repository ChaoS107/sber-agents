1. **Название проекта**

LLM-ассистент — Telegram-бот туристического консультанта  
LLM Travel Assistant Telegram Bot

2. **Описание проекта**

Интеллектуальный LLM-ассистент в формате Telegram-бота, который ведёт естественный диалог с пользователем и выступает в роли профессионального туристического консультанта. Роль и поведение бота задаются в системном промпте LLM: ассистент помогает планировать поездки, подбирать направления, маршруты, отели и активности, а также отвечает на вопросы по визам, сезонности, бюджету и другим аспектам путешествий.

3. **Целевая аудитория**

Частные путешественники и небольшие группы, планирующие самостоятельные поездки: от новичков, которые только начинают путешествовать, до опытных туристов, которым нужны точечные рекомендации и помощь в планировании сложных маршрутов.

4. **Основные функции и возможности**

- Диалоговый режим в Telegram (чаты 1:1 с пользователем)
- Ответы на вопросы о странах, городах, сезонах, требованиях к визам и безопасности
- Подбор направлений по заданным критериям (бюджет, даты, тип отдыха, уровень комфорта)
- Помощь в составлении маршрутов (день за днём с рекомендациями по локациям и активностям)
- Рекомендации по перелётам, отелям, транспорту и развлечениям (без/с интеграциями на следующих этапах)
- Сохранение и обновление выбранных маршрутов пользователя внутри диалога

5. **Технологии и подходы из курса (черновая идея)**

- Использование LLM через API для ведения диалога и реализации роли «профессиональный туристический консультант» через системный промпт
- Возможное применение RAG для ответов на основе собственной базы знаний (справки по странам, визам, локальным правилам)
- Агентные паттерны / workflow-графы для пошагового построения маршрута и уточнения предпочтений пользователя
- MCP / интеграции с внешними сервисами (поиск билетов, отелей, погодных данных) на продвинутых этапах развития проекта
